# 📂 Dicionários para Brute Force

Este diretório contém as wordlists (listas de palavras) utilizadas durante o laboratório de Pentest.

### Conteúdo:
* **usuarios.txt**: Lista com nomes de usuários comuns e contas de serviço padrão.
* **senhas.txt**: Lista de senhas fracas e credenciais conhecidas de fábrica (default credentials).

### Aplicação:
Estas listas foram configuradas para testar a segurança dos serviços **SMB** e **FTP** da máquina alvo Metasploitable 2 através da ferramenta **Medusa**.
